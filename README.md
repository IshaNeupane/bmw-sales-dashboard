# BMW Sales Dashboard

## Overview
This project is an interactive Excel dashboard created to analyze BMW sales performance. It shows revenue, units sold, and sales trends using charts, pivot tables, and slicers.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Features
- Interactive dashboard
- KPI cards
- Revenue analysis
- Sales trend analysis
- Filterable views

## Files in This Repository
- Excel dashboard file
- Dataset file
- Dashboard screenshots

## How to Open
1. Download the Excel file.
2. Open it in Microsoft Excel.
3. Use the slicers and charts to explore the dashboard.

## Purpose
The purpose of this project is to practice dashboard creation and present sales data in a clear and visual way.
